# Welcome to MarketAgent

**MarketAgent** is a professional, scalable command-line interface (CLI) tool for autonomous stock market analysis.

This documentation provides a user guide, an API reference for developers, and instructions for contributing.

## Quick Start

### Installation

```bash
pip install -r requirements.txt
```

### Configuration

Create a `.env` file in the project root:

```env
NVIDIA_API_KEY="YOUR_API_KEY_HERE"
```

### Run Analysis

```bash
python -m app.main analyze AAPL --mock
```

For more details, please see the project [README.md](../README.md).
